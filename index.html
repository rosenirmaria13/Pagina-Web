Bootcamp Diversidade Tech NTT DATA

O Bootcamp NTT DATA Diversidade em Tech traz uma trilha completa para introduzir profissionais em desenvolvimento front-end com HTML, CSS, JavaScript, Angular e AWS.

O programa é para todas as pessoas, com foco em grupos de diversidade.

NTT DATA

A NTT DATA Corporation é uma empresa multinacional japonesa de serviços e consultoria de tecnologia da informação com sede em Tóquio, Japão.

É uma subsidiária parcial da Nippon Telegraph and Telephone.

Evolução Aprendizado HTML

  • O HTML

    • O HTML(Hypertext Markup Language) refere-se a uma linguagem de marcação de base para construção de páginas web.

  • Estrutura básica HTML
    • Quando digitado o comando: "html:5 + tecla TAB" → completa automaticamente a estrutura básica do html. O HTML consiste em duas partes, primeiro a “cabeça” é composta pela tag “head”, e ainda pelo “corpo” pelas tags “body”, que ao serem abertas precisam ser fechadas. Tudo que estiver dentro da tag "html" é HTML.
